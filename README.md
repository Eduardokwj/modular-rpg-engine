# Modular RPG Engine

## 📖 Descrição
Projeto de estudo focado na construção de uma arquitetura modular para jogos 2D em Python utilizando Pygame.

O objetivo não é apenas desenvolver um jogo, mas colocar em prática boas práticas de engenharia de software, demonstrando conceitos como:

- Arquitetura modular
- Separação de responsabilidades
- Game State Machine
- Sistema de entidades
- Sistemas desacoplados
- Estrutura escalável

Este projeto faz parte da minha jornada de transição de carreira para a área de desenvolvimento de software, unindo programação a uma paixão pessoal por jogos.

---

## 🛠 Tecnologias utilizadas

- Python 3.x
- Pygame
- Arquitetura modular
- Programação orientada a objetos

## ▶️ Como executar

### 1. Clonar o repositório

```bash
git clone <url-do-repo>
cd modular_rpg
```

### 2. Criar o ambiente virtual
```bash
python -m venv venv
```
### 3. Ativar o ambiente
```bash
venv\Scripts\activate
```
### 4. Instalar dependências
```bash
pip install -r requirements.txt
```
### 5. Executar o projeto
```bash
python main.py
```
